# 📖ProgressManager

## ✨概要
自分用の進捗確認、引き継ぎメモ用のUnityエディタ拡張です
<br><br>

## 🚀導入方法
Package Manager -> Add package from gitURLにこのレポジトリのURLを入力して追加できます
<br><br>

## 🔧機能(メモ要素)
### ・前回編集していたスクリプト
・前回どのスクリプトを編集していたかを設定できます<br>
・ファイルを設定すると、ボタンからそのスクリプトを開くことができます<br>


### ・前回どこまでやったかのメモ
・前回どこまで何を行ったか、引き継ぎたい内容を文字で記録できます<br>


### ・次に何をするかのメモ
・次に何を行いたいかを、引き継ぎたい内容を文字で記録できます<br>
<br>

## 🔍使い方
1.エディタ立ち上げ時にメモウィンドウが開きます<br>
2.メモウィンドウに引き継ぎたい内容を書き込めます<br>
    ・基本自動保存です<br>
    ・ウィンドウ下の手動保存ボタンからも保存できます<br>
<br><br>

## 🏷️対応バージョン
Unity 2020.1以降のバージョン対応
